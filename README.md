# oomall-system
电子商城系统

### 模块介绍
+ **oomall-common**<br>
整个系统的通用模块，负责响应的包装与异常的统一处理<br><br>
+ **oomall-order**<br>
订单核心模块，负责处理订单业务<br><br>
+ **oomall-seckill**<br>
秒杀模块，负责秒杀活动扣库存<br><br>
+ **oomall-inventory**<br>
库存模块，负责非秒杀活动扣库存<br><br>
+ **oomall-payment**<br>
支付模块，负责模拟支付业务<br><br>
+ **oomall-freight**<br>
运费模块，负责处理运费业务<br><br>
+ **oomall-discount**<br>
折扣模块，负责计算订单折扣<br><br>
+ **oomall-timer**<br>
定时模块，负责商城系统内的定时任务<br><br>
+ **oomall-xxxx-common**<br>
对应模块的通用模块，为对应模块与其他模块之间的通信定义请求和响应对象<br><br>
